# Protein Supplement Market Analysis

This project analyzes protein supplement products from MyProtein Taiwan.

## Objective

Identify market trends and product value using data analysis.

Key questions:
- Is price related to rating?
- Which products are the most popular?
- Which supplements provide the best value?

## Tools

Python  
BeautifulSoup  
Pandas  
Tableau  

## Data Collection

Product data was scraped from MyProtein Taiwan.

Data includes:

- Product name
- Price
- Rating
- Reviews

## Analysis

Three main analyses were performed:

### Price vs Rating
Explores whether higher-priced supplements receive better ratings.

### Popularity Ranking
Ranks products by number of reviews.

### Value Score Ranking
Calculates product value using:

Value Score = (Rating × Reviews) / Price
